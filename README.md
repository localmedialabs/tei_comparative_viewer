# 📖 TEI-IIIF 対照ビューア

このアプリケーションは、**Laravel + Vue 3 + Inertia.js** をベースに構築された、TEI/XML ファイルを解析して HTML 表示する Web アプリケーションです。

## 🚀 技術スタック

- **バックエンド**: Laravel
- **フロントエンド**: Vue 3
- **通信フレームワーク**: Inertia.js
- **ビルドツール**: Vite
